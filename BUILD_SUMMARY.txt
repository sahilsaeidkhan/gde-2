🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════════

Prerequisites:
  1. Node.js 16+ installed
  2. Google OAuth2 credentials (get from Google Cloud Console)
  3. OpenRouter API key (get from openrouter.ai)

Setup (5 minutes):
  
  1. Backend:
     cd backend
     npm install
     cp .env.example .env
     (Edit .env with your credentials)
     npm run dev
     
     ✅ Running on http://localhost:5000

  2. Frontend (new terminal):
     cd frontend
     npm install
     npm run dev
     
     ✅ Running on http://localhost:3000

  3. Chrome Extension:
     chrome://extensions/
     Enable Developer mode
     Load unpacked → select extension/
     
     ✅ Extension loaded

  4. Test:
     Open http://localhost:3000
     Click "Get Started with Google"
     Complete OAuth flow
     
     ✅ You're in the dashboard!
