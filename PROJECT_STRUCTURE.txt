🚀 QUICK START
==============

1. Backend:
   cd backend
   npm install
   cp .env.example .env
   # Edit .env with your credentials
   npm run dev

2. Frontend:
   cd frontend
   npm install
   npm run dev

3. Extension:
   - chrome://extensions/
   - Enable Developer mode
   - Load unpacked → select extension/ folder

4. Test:
   - http://localhost:3000
   - Link Google account
   - Open Google Meet
   - Check extension popup
